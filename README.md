Fancy Ctrl+Z
============

A ZSH plugin that allows you to foreground the last backgrounded job (when you
would normally do `fg`) using `Ctrl+Z`. So in your terminal, you hit `Ctrl+Z`
to background a job, then `Ctrl+Z` again to foreground it.
